# 📚 Book Vibe - Manage Your Reading Journey

**Book Vibe** is a sleek and user-friendly web application designed for book lovers to explore, track, and organize their favorite reads. Users can browse a collection of books, view detailed information, sort them by rating or pages, and maintain their own personal "Read List" and "Wishlist."

## 🚀 Key Features
- **Book Catalog:** Browse a diverse collection of books fetched from a JSON source.
- **Dynamic Routing:** Specific details for every book via React Router.
- **Context API Management:** Centralized state handling for Read Lists and Wishlists.
- **Local Storage Persistence:** Your reading progress is saved locally even after refreshing the page.
- **Sorting Functionality:** Easily sort listed books by **Rating** or **Total Pages**.
- **Interactive UI:** Built with DaisyUI for a clean, responsive, and modern look.
- **Toast Notifications:** Real-time feedback using `react-toastify` for adding or duplicating books.

---

## 🛠️ Technology Stack
- **Core:** React 19
- **Routing:** React Router v7
- **Styling:** Tailwind CSS (v4) & DaisyUI
- **Icons:** React Icons
- **State Management:** React Context API
- **Data Persistence:** Browser Local Storage
- **UI Components:** React Tabs

💻 How to Run Locally
Follow these steps to get the project running on your local machine:

Clone the repository:

Bash
git clone [https://github.com/your-username/book-vibe.git](https://github.com/your-username/book-vibe.git)
Navigate to the project folder:

Bash
cd book-vibe
Install dependencies:

Bash
npm install
Start the development server:

Bash
npm run dev
Open in browser:
Go to http://localhost:5173 to see the app in action.
